Benchmark of PYSCAL_GLASSVIEWER¶

Pyscal_Glassviewer is a FORK of Pyscal, which is modified and improved for Metal Glass data post-processing tasks by Zihao Bai, Jilin University. This Benchmark uses a POSCAR file from MD of Al melting process. In order to test the partial PDF module and SRO module, we label the atoms into Al1 and Al2. The properties is as following:

Box parameter:

55.996 0.0 0.0
0.0 56.923 0.0
0.0 0.0 56.369

Atom Numbers:

Al1 Al2
10000 976
